# Visual Scheduler 視覺化排程工具 - 使用手冊

本工具是一款基於 Web 的甘特圖排程應用程式，旨在幫助專案經理和團隊成員輕鬆規劃、追蹤和管理專案進度。

## 核心功能

### 1. 專案管理
- **建立新專案**：點選左上角「File」選單中的「New Project」可清空當前資料並開始新專案。
- **開啟專案**：點選「Open Project」可匯入先前儲存的 `.gantt` (或 `.json`) 專案檔案。
- **儲存專案**：點選「Save Project」可將當前專案狀態匯出為 `.gantt` 檔案，方便日後讀取或分享。
- **專案設定**：點選右上角的「專案設定」圖示，可修改客戶名稱、專案名稱及版本號。

### 2. 任務編輯與操作
- **新增任務**：點選左側側邊欄頂部的「+」按鈕。
- **編輯任務**：在側邊欄雙擊任務名稱，或在甘特圖中雙擊任務條，即可開啟編輯視窗。
- **調整時間**：在甘特圖中，您可以直接拖拽任務條來移動時間，或拖拽左右邊緣來縮放任務期間。
- **調整進度**：在任務編輯視窗中調整進度百分比，任務條會以深色顯示完成部分。
- **里程碑設定**：在編輯視窗中勾選「里程碑」，任務將轉變為菱形標記，代表關鍵節點。
- **任務排序**：在左側側邊欄，您可以透過拖拽任務左側的圖示來調整任務的上下順序。

### 3. 圖表檢視與縮放
- **視圖切換**：提供「日視圖」與「週視圖」切換，適合不同精細度的排程需求。
- **縮放功能**：使用工具列的「放大/縮小」按鈕，或按住 `Ctrl` (或 `Cmd`) 鍵並滾動滑鼠滾輪來調整圖表比例。
- **格線與標記**：在「設定」中可開啟/關閉垂直格線，以及今日日期的背景高亮標記。

### 4. 版本控制與復原
- **儲存版本**：在「專案設定」中可以儲存當前狀態為一個版本，並附帶修改描述。
- **還原版本**：隨時可以從版本履歷中選擇舊版本進行還原。
- **復原/重做 (Undo/Redo)**：工具列提供復原與重做按鈕，可快速撤銷或恢復最近的操作。

### 5. 匯出與列印
- **匯出圖片 (SVG)**：將當前甘特圖匯出為高品質的 SVG 向量圖檔。
- **列印 (Print)**：直接調用瀏覽器列印功能，將排程表列印成 PDF 或紙本。

## 快捷鍵與操作技巧
- **雙擊任務名稱**：快速編輯任務。
- **拖拽任務條**：快速調整排程。
- **Ctrl + 滾輪**：快速縮放圖表。
- **自動儲存**：系統會自動將您的變更保存在瀏覽器本地，即使重新整理頁面也不會遺失資料。
